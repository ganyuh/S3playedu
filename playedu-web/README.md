# Vue 3 + Vite

# 项目结构

``` dir
├─public
└─src
    ├─assets # 静态资源目录
    │  └─css # 公用 css
    ├─components # 共组件
    ├─config # 配置目录
    ├─hooks # 钩子目录
    ├─router # 路由配置目录
    ├─service # API 目录
       └─modules # 一个实体类对应一个文件
    ├─stores # Pinia Store 文件
    │  └─modules 
    ├─utils
    └─views # 页面文件
        ├─backstage # 后台组件
        └─frontdesk # 前台组件
```
